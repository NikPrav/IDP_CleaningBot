# Cleaning Bot

Code for the cleaning bot that was made for IDP in the year 2019-20. Has two folders, BotAlgo which is the basic algorithm for the bot the traverse the whole room, and the BotInferface which has the code for the interface + arduino code

## BotInterface

Has two parts : A WebApp which has the controls and calls a Blynk API to update values in the Blynk server, and the arduino code which fetches the values from the Blynk server realtime and moves the Bot. <br />

[Blynk HTTP RESTful API](https://blynkapi.docs.apiary.io/#)

<br />

## BotAlgo

Simple algorithm to traverse the whole rorm while avoiding obstacles
